# Telegram AI Bot SkillUP
# Команда Пюрешка

## Технологии
- Docker
- Docker Compose
- Python 3.12
- Pydantic
- Aiogram
- PostgreSQL
- Redis

## Настройка
1. Скопируйте `.env.example` и `example_ai_config.yaml`.
2. Переименуйте их в `.env` и `ai_config.yaml` соответсвенно.
3. Заполните конфигурационные файлы.

## Запуск приложения
```bash
  docker-compose up --build
```
